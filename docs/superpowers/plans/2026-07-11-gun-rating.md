# Gun-Rating Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** When the enemy fields armor, make the spawn picker prefer the highest armor-penetration unit available within the tier it already chose.

**Architecture:** An offline Python tool extracts each unit's AP penetration from the RobZ paks into a generated Lua table (`gun_ratings.lua`). The bot loads that table and, inside the existing within-tier candidate weighter, multiplies a unit's draw weight by a bounded factor derived from its penetration, but only when `EnemyHasTanks()` is true. Additive and nil-safe: no table, or no enemy armor, means stock behavior.

**Tech Stack:** Python 3 (stdlib `zipfile`, `re`) for the offline tool; Lua 5.x for the bot and its offline test harness.

## Global Constraints

- AP-only. No HE, no anti-infantry, no range term.
- Army-wide. The boost applies to every combat spawn (waves, backfill, group fills), not only the main group.
- Gun rating for a unit = the MAX, over all of that unit's weapons, of the weapon's default-AP curve peak band `a(...)` value (integer mm). APCR/HEAT/FG ammo variants are ignored.
- Picker constants, exact: `GunRatingRef = 60`, `GunRatingMulMin = 0.5`, `GunRatingMulMax = 1.8`.
- Gate: boost applies only when `BotApi.Commands:EnemyHasTanks()` returns true.
- Boost multiplier is `clamp(rating / GunRatingRef, GunRatingMulMin, GunRatingMulMax)`; never zero, never bans a unit.
- `gun_ratings.lua` is a generated, committed artifact (same treatment as `flag_sectors.lua`). Missing key => nil => no-op.
- No change to any spawn count, cap, cadence, tier ratio, `DecideTier`, `retire`, bias floor, or unlock/retire time.
- Known-good anchor values from the real RobZ paks: `sdkfz222 = 46`, `m5a1 = 78`, `pz3_m = 97`.
- RobZ pak location: `/mnt/storage/steam/steamapps/common/Men of War Assault Squad 2/mods/robz realism mod 1.30.10/resource/` (`gamelogic.pak`, `entity.pak`).
- Lua tests run: `cd resource/script/multiplayer && lua tests/<name>_spec.lua`. Compile check: `luac -p bot.lua && luac -p bot.data.lua`.

Notes for the implementer:
- The default-AP curve is the FIRST `("damage_NNN" a(...) ...)` at the top level of the
  resolved gun text, NOT the one inside a `{parameters "apcr" ...}` / `"heat"` / `"fg"`
  block. Read the value inside `a( )`.
- `{from "PARENT"}` may point at another gun file OR at `"pattern gun"` (a base pattern
  with no AP curve of its own). Stop the walk when you either find an AP curve or hit a
  parent that is not a member of `gun_files`.
- A unit's `.def` may live under any `entity/-vehicle/**/` subtree and its basename is not
  always the unit id (e.g. `pz3_m` lives at `.../tank_medium/pz3n/pz3_m.def`). Match the
  `.def` whose basename equals `<unit_id>.def`.
- A `.def` can list several `{weapon "..." ...}` lines (main gun + coax MG); the max rule
  makes the main gun win.

```python
"""Generate gun_ratings.lua: unit_id -> AP peak penetration (mm) from RobZ paks.

Rating = max over a unit's weapons of the weapon's default-AP curve peak band a().
AP only; APCR/HEAT/FG ignored. Run from repo root: python tools/build_gun_ratings.py
"""
import os
import re
import sys
import zipfile

# top-level default-AP curve: first ("damage_NNN" a(NN) ...) not inside a {parameters ..} block.
_AP_A = re.compile(r'\("damage_\d+"\s+a\((\d+)\)')
_FROM = re.compile(r'\{from\s+"([^"]+)"')
_WEAPON = re.compile(r'\{weapon\s+"([^"]+)"')

def _strip_param_blocks(text: str) -> str:
    """Remove {parameters "apcr"/"heat"/... ( ... )} blocks so their a() is not matched."""
    out, depth, i, keep = [], 0, 0, True
    # simple brace-scan: drop any {parameters ...} span
    tokens = re.split(r'(\{parameters\b|\{|\})', text)
    skip_depth = None
    depth = 0
    for tok in tokens:
        if tok == '{parameters':
            if skip_depth is None:
                skip_depth = depth
            depth += 1
            continue
        if tok == '{':
            depth += 1
        elif tok == '}':
            depth -= 1
            if skip_depth is not None and depth == skip_depth:
                skip_depth = None
                continue
        if skip_depth is None:
            out.append(tok)
    return ''.join(out)

def resolve_ap_peak(name, gun_files, _seen=None):
    _seen = _seen or set()
    if name in _seen or name not in gun_files:
        return None
    _seen.add(name)
    text = gun_files[name]
    m = _AP_A.search(_strip_param_blocks(text))
    if m:
        return int(m.group(1))
    fm = _FROM.search(text)
    if fm:
        return resolve_ap_peak(fm.group(1), gun_files, _seen)
    return None

def gun_rating_for_weapons(weapons, gun_files):
    best = 0
    for w in weapons:
        v = resolve_ap_peak(w, gun_files)
        if v and v > best:
            best = v
    return best

def _read_gun_files(gamelogic_path):
    out = {}
    with zipfile.ZipFile(gamelogic_path) as z:
        for n in z.namelist():
            if "/set/stuff/gun/" in n and not n.endswith("/"):
                out[os.path.basename(n)] = z.read(n).decode("latin-1", "replace")
    return out

def _read_unit_ids(gamelogic_path):
    ids = set()
    unit_re = re.compile(r'\{"([A-Za-z0-9_]+)"\s+\("v"')
    with zipfile.ZipFile(gamelogic_path) as z:
        for n in z.namelist():
            if "/set/multiplayer/units/" in n and n.endswith(".set"):
                for m in unit_re.finditer(z.read(n).decode("latin-1", "replace")):
                    ids.add(m.group(1))
    return ids

def _read_defs(entity_path):
    """basename(without .def) -> list of weapon names."""
    out = {}
    with zipfile.ZipFile(entity_path) as z:
        for n in z.namelist():
            if "/-vehicle/" in n and n.endswith(".def"):
                base = os.path.basename(n)[:-4]
                text = z.read(n).decode("latin-1", "replace")
                out[base] = _WEAPON.findall(text)
    return out

def build_ratings(gamelogic_path, entity_path):
    gun_files = _read_gun_files(gamelogic_path)
    defs = _read_defs(entity_path)
    ratings = {}
    for uid in _read_unit_ids(gamelogic_path):
        weapons = defs.get(uid)
        if not weapons:
            continue
        r = gun_rating_for_weapons(weapons, gun_files)
        if r > 0:
            ratings[uid] = r
    return ratings

def _emit_lua(ratings):
    lines = ["-- GENERATED by tools/build_gun_ratings.py against RobZ 1.30.10.",
             "-- unit_id -> AP peak penetration (mm). Do not edit by hand.",
             "return {"]
    for uid in sorted(ratings):
        lines.append('    %s = %d,' % (uid, ratings[uid]))
    lines.append("}")
    return "\n".join(lines) + "\n"

def main():
    robz = os.environ.get("ROBZ_RESOURCE",
        "/mnt/storage/steam/steamapps/common/Men of War Assault Squad 2/"
        "mods/robz realism mod 1.30.10/resource")
    gl = os.path.join(robz, "gamelogic.pak")
    en = os.path.join(robz, "entity.pak")
    ratings = build_ratings(gl, en)
    here = os.path.dirname(os.path.abspath(__file__))
    out = os.path.join(here, "..", "resource", "script", "multiplayer", "gun_ratings.lua")
    with open(out, "w") as f:
        f.write(_emit_lua(ratings))
    print("wrote %d ratings to %s" % (len(ratings), os.path.normpath(out)))

if __name__ == "__main__":
    main()
```

  (does not fully close) the German 460-530s Stuart window by adding the Puma at 530s.

  makes every lookup nil and restores stock picker behavior; the Puma rows are inert data.
