# Home-defense Sectors Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make tier-1 home defense fire for both teams by fixing the offline sector data, so the GER bot retakes its home flags.

**Architecture:** All changes are in the offline generator `tools/build_sectors.py`. It deduplicates cross-team base tags, trims each side's base set to an equal count, then applies a per-team two-point axis renormalization anchored to base flags. The regenerated `flag_sectors.lua` makes each team's deepest base flag classify as OWN at runtime. `bot.lua` is not modified.

**Tech Stack:** Python 3 (offline tool), Lua 5.x (runtime + offline specs).

def _side_dist(flags, bases, name, side):
    ds = [_dist(flags[name], bp) for bn, bp in bases.items() if bn[0] == side]
    return min(ds) if ds else float("inf")
```

    base = {n: set() for n in names}
    for bname, bp in bases.items():
        dists_to_base = sorted((_dist(flags[n], bp), n) for n in names)
        close = {n for d, n in dists_to_base if d < THRESH}
        for _d, n in dists_to_base[:KFLOOR]:

            base[n].add(bname[0])
    # dedupe: a flag tagged for both teams keeps only its nearer side (a wins exact ties)
    for n in names:
        if {"a", "b"} <= base[n]:
            da = _side_dist(flags, bases, n, "a")
            db = _side_dist(flags, bases, n, "b")
            base[n] = {"a"} if da <= db else {"b"}
    # trim: keep N = min(countA, countB) nearest flags per side
    a_flags = sorted((n for n in names if base[n] == {"a"}),
                     key=lambda n: (_side_dist(flags, bases, n, "a"), n))
    b_flags = sorted((n for n in names if base[n] == {"b"}),
                     key=lambda n: (_side_dist(flags, bases, n, "b"), n))
    N = min(len(a_flags), len(b_flags))
    assert N >= 1, "a side or b side has no base flag"
    keep = set(a_flags[:N]) | set(b_flags[:N])
    for n in names:
        if base[n] and n not in keep:
            base[n] = set()

def renorm(comp, adj):
    """Per-team two-point axis renormalization anchored to base flags.
    homeA = min raw axis over a-base flags; homeB = max raw axis over b-base flags.
    Returns {name:(x,y,axis')} with axis' = clamp01((axis-homeA)/(homeB-homeA))."""
    a_axes = [comp[n][2] for n in comp if "a" in adj[n][1]]
    b_axes = [comp[n][2] for n in comp if "b" in adj[n][1]]
    if not a_axes or not b_axes:
        raise SystemExit("renorm: a side or b side has no base flag")
    homeA, homeB = min(a_axes), max(b_axes)
    if not (homeA < homeB):
        raise SystemExit("renorm anchors crossed: homeA=%.3f >= homeB=%.3f" % (homeA, homeB))
    span = homeB - homeA
    out = {}
    for n, (x, y, axis) in comp.items():
        r = (axis - homeA) / span
        out[n] = (x, y, 0.0 if r < 0 else 1.0 if r > 1 else r)
    return out
```
